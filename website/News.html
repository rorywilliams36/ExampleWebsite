Record number of users on the new and improved website


The Puzzle Room is an website which has be gaining an unexpected amount of popularity in the recent months. The founder, Rory Williams has stated 'The increasing popularity has been caused by the implementation of our new game". The website has seen an increase from an average of 2000 users a month has doubled to 4000, with it peak at 10,000 users at one time which was just over 2 months ago.


This has also caused other rival websites to start popping up which have been following the same success using the same idea. The competition have been seeing mixed but mostly positive results.


The popularity gained has been staggering for such a small website with an evenre smaller team. The website started last year and wasn't able to get 10 different users to use the website for a couple of months. The founder Williams stated that the development team has been working tirelessly and hoping that the website would gain traction somehow.
